# CS50 Week 0 - Scratch Projects

## Project Rabbit

A game inspired by a childhood memory. In grade LKG, I played a bird-catching-fish game at a friend's house and loved it. Years later, I decided to recreate it using Scratch.

**What the game does:**
- Control a bird character to catch falling fish
- Earn points for each fish caught
- Simple but engaging gameplay

**What I learned:**
- Explored all of Scratch's code elements (loops, conditionals, events, sprites)
- Understood how to structure game logic
- Learned collision detection and scoring systems
- Practiced iteration and improvement

**Status:** Version 1 complete. Version 2 in progress with smoother gameplay and better game experience.

**How to play:**
1. Open the project in Scratch
2. Click the green flag to start
3. Use arrow keys or mouse to control the bird
4. Catch as many fish as possible!

## Reflection

This project taught me that programming isn't just about following instructions—it's about solving problems creatively. I had to figure out how to recreate a game I only half-remembered, using tools I was learning for the first time. That process of exploration, failure, and improvement is what makes coding exciting.
